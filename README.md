 MNIST Digit Classification Using Deep Learning

 Project Overview
This project uses a Deep Neural Network (DNN) built with TensorFlow/Keras to classify handwritten digits from the MNIST dataset.

 Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

 Dataset
MNIST handwritten digit dataset.

 Project Workflow
1. Import Libraries
2. Load MNIST Dataset
3. Data Preprocessing
4. One-Hot Encoding
5. Build Neural Network
6. Train Model
7. Evaluate Performance
8. Generate Predictions
9. Confusion Matrix Visualization

 Results
The model successfully classifies handwritten digits with high accuracy.
